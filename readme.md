# 🍳 Minha Cozinha - Culinária Inteligente

Organize seus ingredientes, descubra receitas e aproveite seus créditos de ICMS!

## ✨ Funcionalidades

- **📸 Upload de Nota Fiscal**: Envie a foto da sua nota e os itens serão adicionados automaticamente ao estoque
- **📦 Gestão de Estoque**: Visualize todos os itens que você tem em casa
- **🍽️ Sugestão de Receitas**: Descubra o que cozinhar com os ingredientes disponíveis
- **💰 Créditos ICMS**: Selecione seu estado e seja redirecionado ao programa oficial de créditos

## 🚀 Tecnologias

- **Backend**: Python + Flask
- **Frontend**: HTML + Tailwind CSS + JavaScript puro
- **Banco de Dados**: SQLite (planejado para migrar para PostgreSQL)
- **Deploy**: Render (gratuito)
